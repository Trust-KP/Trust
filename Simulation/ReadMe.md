# Simulation Models

This directory contains the simulation models used in the marine insurance institutional experiments.

Each simulation extends the previous model by introducing additional mechanisms governing underwriting behaviour, financial structure, market interaction, or capital allocation.

---

## Model Progression

| Simulation | Focus |
|-----------|-------|
| SIM A | Basic underwriting behaviour |
| SIM B | Underwriting doctrine |
| SIM C | Core financial engine |
| SIM D | Institutional financial mechanics |
| SIM E | Shared-risk market participation |
| SIM F | Market coordination mechanisms |
| SIM G | Capital regime experimentation |
| SIM H | Capital scaling dynamics |

---

## Directory Structure

Each simulation folder contains two types of files.

### Specification

Human-readable documentation describing the conceptual structure of the model.

Example:
SIM_F/SIM_F_Specification.md

### Implementation

Executable code implementing the simulation.

Example:
SIM_F/sim_f_engine.py

---

## Notes

These models are exploratory simulations designed to illustrate institutional dynamics within stylised insurance markets.

They are not calibrated to represent any specific real-world market.
